# Full-Stack Development with Django and React.js ( Learning Management System )
* This project is being developed as a learning experience to deepen my understanding of full-stack development.
 The focus is on building a web application with a React ( frontend ) and Django ( backend ), utilizing Django RESTful APIs to enable communication between frontend and backend.
 Through this project, I will gaining hands-on experience in building, integrating and developing a full-stack application, while enhancing my skills in both frontend and backend technologies.
 This project serves as a practical exploration of modern web development practices and tools.

# Features I will develop
* real-world fullstack application
* Created REST-API
* Built an email system for user registration and password reset via email
* build a robust authentication and course management system
* the system will handle thousands of courses
* students will be able to add cources to the cart
* preview cart
* enter coupons to get discounts
* checkouts by doing real payment 
* checkout
* upload videos
* downloading files
* chat management system
* View students and their progress
* get notifications
* Implement email sending functionality using Django
